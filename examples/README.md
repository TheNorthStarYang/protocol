# English

## Examples

This directory contains sample materials for developers building on top of the public AhaKey protocol.

The goal of this directory is not to provide every possible integration, but to make it easier to build:

- third-party desktop clients
- debugging tools
- protocol experiments
- workflow integrations
- small developer utilities

---

## What belongs here?

Examples in this directory may include:

- minimal BLE scanning examples
- connection and discovery examples
- notification subscription examples
- state query examples
- command write examples
- image-related experimental flow examples
- debugging snippets
- sample parsers

---

## Stability note

Not every example here should be treated as production-ready.

Examples may demonstrate:

- stable protocol paths
- experimental protocol paths
- debugging-oriented flows

Please read the documentation in `docs/` first, especially:

- `docs/overview.md`
- `docs/stability-policy.md`
- `docs/ble-services.md`
- `docs/events-and-states.md`
- `docs/commands.md`

---

## Current starting point

If you are new to AhaKey protocol development, start with:

- `minimal-flow.md`

That document describes the safest minimal connection and command flow for a third-party client.
